# Prompt Builder for Cursor

Расширение для браузера, которое помогает быстро формировать структурированный промт для ИИ (например, Cursor), копировать его в буфер обмена и хранить шаблоны для повторного использования.

## 1) Что это за плагин и для чего

`Prompt Builder for Cursor` - это popup-расширение с удобной формой, где вы заполняете блоки промта (`role`, `task`, `context` и другие), а затем копируете готовый текст в нужном формате с тегами.

Плагин полезен, когда нужно:

- стандартизировать запросы к ИИ;
- быстрее собирать качественные промты;
- хранить готовые варианты и переиспользовать их;
- редактировать уже сохраненные шаблоны без ручного копирования.

## 2) Как использовать основной функционал

Во всплывающем окне есть две вкладки:

- `Промт` - форма для ввода и генерации промта;
- `Сохраненные` - список сохраненных шаблонов.

### Вкладка `Промт`

Доступные поля:

- `Тема запроса (название)` - краткое имя шаблона; обязательно для сохранения.
- `<role>` - роль, в которой должен отвечать ИИ (например, senior frontend-разработчик).
- `<task>` - конкретная задача, которую нужно выполнить.
- `<context>` - исходные условия, ограничения проекта, стек и важные детали.
- `<success_criteria>` - критерии, по которым можно понять, что ответ получился успешным.
- `<example>` - пример желаемого формата/содержания ответа.
- `<audience>` - целевая аудитория ответа (junior, PM, клиент и т.д.).
- `<priority>` - приоритет ответа (например, скорость, качество, точность, простота).
- `<validation>` - флаг самопроверки перед финальным ответом (`true/false`).
- `<constrains>` - что делать не нужно, запреты и исключения.
- `<output>` - формат итогового результата (список, JSON, шаги, код и т.д.).

Кнопки:

- `Копировать промт` - копирует собранный промт в буфер обмена (пустые поля не добавляются).
- `Сохранить промт` - сохраняет текущий шаблон.

### Вкладка `Сохраненные`

- Можно хранить до **5** промтов.
- Если лимит достигнут, сохранение блокируется и показывается сообщение.
- Для каждого сохраненного промта доступны действия:
  - просмотр содержимого;
  - копирование содержимого в память;
  - редактирование (открывает промт в первой вкладке);
  - удаление без подтверждения.

Дополнительно:

- все состояния и сохраненные промты хранятся локально (`chrome.storage.local`);
- если сохранить промт с уже существующим названием, происходит перезапись существующего.

## 3) Как установить в браузер

### Google Chrome

1. Откройте страницу `chrome://extensions`.
2. Включите `Режим разработчика`.
3. Нажмите `Загрузить распакованное расширение`.
4. Выберите папку проекта (где лежит `manifest.json`).
5. Закрепите иконку расширения на панели (по желанию).

### Microsoft Edge

1. Откройте страницу `edge://extensions`.
2. Включите `Режим разработчика`.
3. Нажмите `Загрузить распакованное`.
4. Выберите папку проекта с `manifest.json`.

После установки нажмите на иконку расширения в панели браузера, чтобы открыть popup.
